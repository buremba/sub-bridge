Setup in minutes

Install once, connect your subscription, and point Cursor at your local MCP proxy.

1

Install the MCP server in Cursor

Click the install button or add a new MCP server with npx -y sub-bridge.

2

Ask Cursor to connect

In Cursor chat, run get_connection. It returns a local URL and login links for ChatGPT or Claude.

3

Log in to your subscription

Open the local URL, sign in with ChatGPT or Claude, and grant access. Tokens stay on your machine.

4

Paste the URL and token into Cursor

Cursor Settings -> Models -> API Keys. Set Base URL to https://your-public-url/v1 and API key to your OAuth token.

5

Use your models

Select the model in Cursor and start coding. Autocomplete still requires a Cursor plan.

What you see in the local UI

The local page shows your public URL, login buttons, and the API key you paste into Cursor models settings.

How it works

Sub Bridge runs a local MCP server, opens a tunnel for Cursor, and proxies requests to your subscription provider.
